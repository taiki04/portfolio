デイトラ「sobolon」

  • 制作時期：2020年11月
  • 制作期間：３日間
  • 主な使用ツール・言語：HTML / CSS / Sass / Javascript (jQuery) / Adobe XD

デザインカンプ（XD）からコーディングをしました。
ハンドメイドアクセサリーのLPで、とても女の子が好きそうな色合いのサイトです。
コーディングをしていて意識したところはフォーム周りに関してです。チェックボックスに動きをつけたり、TAB操作をしていることが視覚的にわかるように :focus を用いて色を付けたりと、アクセシビリティにも考慮しました。

デイトラ「Sample corp」

  • 制作時期：2020年11月
  • 制作期間：7日間
  • 主な使用ツール・言語：HTML / CSS / Javascript (jQuery) / Adobe XD

デザインカンプ（XD）からコーディングをしました。
制作会社のLPで、黒色と白色の２色構成で作られており力強い男性的なサイトです。
コーディングをしていて意識したところは、背景色に黒を用いてその上に構成されているコンテンツ部分を置き、微妙にずらしたりしている部分を再現するところです。positionプロパティを意識してコーディングを行う場面や、ネガティブマージンを使用しコンテンツの位置をずらしたりしました。
